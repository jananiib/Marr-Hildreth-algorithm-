# Edge detection

C++ Qt implementation of edge detection algorithms.

Currently available algorithms:

 - Canny edge detector
 - Sobel edge detector
 - Prewitt edge detector
 - Roberts cross
 - Scharr operator

The project includes GUI for viewing results.
